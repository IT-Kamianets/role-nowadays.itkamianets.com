# True Face Mode

True Face Mode is a deduction game where players try to discover the **secret roles** assigned to every participant. Each player receives a hidden role at the beginning of the game. During each round, players attempt to guess the roles of others from a set of possible options.

Instead of eliminating players, the goal is to **correctly identify all roles** in the game using logic, observation, and deduction.

---

# Players

Minimum players:

**1**

Maximum players:

**20**

This mode supports both solo and multiplayer play.

---

# Game Setup

At the start of the game:

1. The system assigns a **secret role** to every player.
2. Roles remain hidden from all other players.
3. Each player only sees their **own role**.

Roles can be selected from a predefined pool depending on the game configuration.

---

# Game Rounds

The game proceeds in rounds.

Each round follows this flow:

1. Players choose what role they believe each participant has.
2. When all players submit their guesses, the round is resolved.
3. The game reveals **how many roles were guessed correctly**.

Important:

- The system does **not reveal which guesses were correct**.
- Only the **total number of correct guesses** is shown.

---

# Example

Players: 4

Secret roles:

- Player A → Knight  
- Player B → Healer  
- Player C → Assassin  
- Player D → Defender  

A player submits guesses.

Result shown:

**2 roles correctly identified**

Players must use this information to refine their guesses in the next round.

---

# Logical Rounds Limit

Each game has a **logical round limit**, based on the number of players and roles.

If players correctly identify all roles **within the logical round limit**, the game is considered:

**Success**

If the logical round limit is exceeded before all roles are identified:

**Fail**

This encourages efficient deduction rather than random guessing.

---

# Victory Condition

The game ends when:

- All roles are correctly identified.

or

- The logical round limit is exceeded.

---

# Purpose of the Mode

True Face Mode focuses on:

- logical deduction
- pattern recognition
- strategic guessing
- collaborative reasoning

The mode can be played competitively or cooperatively.
